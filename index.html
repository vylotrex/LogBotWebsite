How it works

1) Setup the Bot

Easy installation using the /setup command.

2) Listen to Voice Channel Events

LogBot tracks VoiceChannel joins, leaves and moves between channels.

3) Send clean embeds

Log channels receive tidy embeds. With all important information.

Why LogBot?

Everything you need to keep voice-channel activity transparent and auditable.

Voice-Channel Join/Leave/Move

Accurate logs with timestamp and user/channel context.

Guided Setup

Use /setup to pick separate channels for Join, Leave and Move logs OR use an all-in-one channel.

Searchable History

Use /search to list recent activity for a specific member.

Cleanup

Admins can /clear logs (for a single user or server-wide).

About the Developer

Hey! I’m draxylith, and LogBot is my first public project.

This bot isn’t crammed with endless features — and that’s intentional.
